# **App Name**: Lecture Scribe

## Core Features:

- Audio Cleaning: Process uploaded audio to reduce background noise and irrelevant speech.
- Speaker Identification: Differentiate speech segments from the teacher and other sources in the audio.
- Transcription: Transcribe the teacher's speech into text.
- Topic Extraction: Use an LLM tool to extract main topics from the transcription.
- Note Generation: Automatically create summarized notes for each topic.
- Downloadable Notes: Provide downloadable notes in Markdown or PDF format with timestamps.
- File Upload: Allow users to upload audio files directly through the web app.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey knowledge and professionalism.
- Background color: Light gray (#F0F0F5), very slightly tinted blue, for a clean, unobtrusive feel.
- Accent color: Light blue (#7986CB), for interactive elements.
- Body and headline font: 'Inter', a sans-serif font for clear and accessible readability.
- Use simple, outline-style icons to represent topics and functionalities.
- Clean, single-column layout for easy navigation and focus on content.
- Subtle loading animations and transitions when processing audio or generating notes.
